# ROBOTS.md

## Purpose
This project takes student rankings of capstone projects to assign students to a project based on their ranking preferences. Each project also has student capacity constraints and each student has the constraint of only being assigned one project.

## Allowed
- Index public pages for search engines
- Access documentation under `/docs/`
- Use content snippets for summaries with attribution

## Disallowed
- Scraping user data or personal information
- Accessing `/private/` or `/admin/` routes
- Training AI models on proprietary content without explicit permission

## Rate Limits
- Maximum 1 request per minute
- Respect server load and back off on errors
